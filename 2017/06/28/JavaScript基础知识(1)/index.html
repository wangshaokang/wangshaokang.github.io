JavaScript是什么?

js是一门弱类型的脚本语言 是一门 解释型 语言
【html和css是文本语言，用来描述的】

JS是运行在用户端（客户端） 浏览器上的语言
JS运行在客户端 容易被篡改 所以安全性较低

课后思考:
防篡改 负载均衡（独立的均衡）
API 百度百科

ECMAScript的版本： es5 和 es6版本

###DOM 文档对象模型 (document object model)

#####DOM操作的是什么？
文档对象（document）【html文件中的所有文件内容】
标签：由尖括号和关键字组成
元素：成对的标签和内容组成（内容在元素中可以为空）

root Element(根元素)
根目录

###BOM 浏览器对象模型(browser object model)

####BOM是操作什么的？
window对象 （ 浏览器的顶层对象）

##JS的引入方式

  • 行内式：将js代码直接写在标签内
    <input type=""button" onclick="alert('zhufengpeixun')"
  • 内嵌式：将js代码块写在<script></script>中，放在html文件中
  • 外链式：通过<script src="url"></script>引入一个外部JS文件

    js引入的时候一般放在html所有的内容之后

对象的属性

  • 内置属性：天生自带的
  • 自定义属性：人为设定的
    对象的属性是用来描述对象的，还可以存储值

##JS中用来存储值得方式

  • 变量
  • 属性

###对象的属性名可以为纯数字

#####访问对象的方式

  • 对象名.属性名
  • 对象名[‘属性名’]
    在获取对象属性的时候，如果属性名[‘属性名’]

#####增加属性的方式
自定义属性的过程，增加的时候需要增加属性名和属性值

  • 对象名.新的属性名=属性值
  • 对象名[‘新的属性名’]=属性值
    #####删除属性的方式
    delete 对象名.属性名
    delete 对象名[属性名]

#####修改属性值的方法
就是将之前有的属性名里面的属性值重新赋值
对象名.属性名=新的属性值
对象名.[‘属性名’]=新的属性值

属性名为数字的时候，可以省略引号，其他属性名省略引号，系统默认这个属性名为变量
