##flexbox 弹性的盒模型

###flex-direction 方向(父级元素)

flex-direction：
主轴与行内轴方向作为默认的书写模式。即横向从左到右排列（左对齐）。
row-reverse：
对齐方式与row相反。
column：
主轴与块轴方向作为默认的书写模式。即纵向从上往下排列（顶对齐）。
column-reverse：
对齐方式与column相反。

flex-wrap：
flex容器为单行。该情况下flex子项可能会溢出容器
wrap：
flex容器为多行。该情况下flex子项溢出的部分会被放置到新行，子项内部会发生断行
wrap-reverse：
反转 wrap 排列。

###flex-flow(flex-direction,flex-wrap)

flex-flow（row，nowrap）
弹性盒子里的子集盒子的流动布局方式（水平方向从左到右，盒子平铺整个一行 不折行）
flex-flow（row，wrap）
弹性盒子里的子集盒子的流动布局方式（水平方向从左到右，盒子平铺整个一行多出的部分可以折行）
flex-flow（row，wrap-reverse）
弹性盒子里的子集盒子的流动布局方式（水平方向从左到右，反转整个盒子的排布{从左下角开始，向右平铺，然后这折行这一行的左上角继续平铺}

###justify-content:center; 水平居中

###align-items:center 垂直居中

###align-content 跟align-items属性值比较相似（多行)

##4个兼容写法
display:flex;
display:-webkit-box;
display:-webkit-flex;
display:-ms-flexbox;

-webkit-box-orient:vertical;
表示子元素在父元素里面的排列方向
-webkit-box-pack:justify;
表示子元素的对齐方式

flex-direction:column-reverse;
-webkit-flex-direction:column-reverse;
主轴与行内轴方向作为默认的书写方式。即横向从左到右排列（左对齐）

##flex-wrap
flex-wrap:wrap;
-webkit-flex-wrap:wrap;
flex容器为多行，该情况下子项可能会溢出容器

flex-wrap:nowrap;
-webkit-flex-wrap:wrap
flex容器为单行，该情况下flex子项可能会溢出容器

flex-wrap:wrap-reverse;
反转wrap排列
