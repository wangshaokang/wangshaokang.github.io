##基本数据类型

##引用数据类型
1、浏览器提供给代码运行的环境，代码在这个里从上到下的执行
2、当遇到引用数据类型的时候，浏览器会开辟一个新的内存空间，将引用数据类型中的代码当做字符串存到这里面
3、开辟的空间会返回给对象一个地址，那么此时这个对象存储的不是代码而是一个地址
4、当将这个对象赋值给另一个对象的时候，其实赋值过程是将地址传递给另一个对象
也就是说，这两个对象将要访问同一个地址，也就是说他们操作的是同一分代码

##数据类型之间的比较

  • 对象==对象 永远是false
    • ({}=={}) ([]==[])
  • 数字==字符串 字符串调用Number()方法，将字符串转为数字，然后在进行比较
  • 数字==布尔 布尔转为数字类型数据，调用Number()
    true=1 false=0

-数字==对象 对象调用.toString()变成字符串，字符串调用Number()方法

只要两侧有引号的字符都是字符串，字符串在JS中没有意义

null 预留一个位置
undefined 本应有值 但是没有

**对象和其他数据类型比较的时候，首先都要转为字符串

最终比较的时候，其实都是数字在比较

###规律
1、对象==对象 false
2、对象==字符串 对象转为字符串
3、对象==数字 对象转为字符串，字符串转为数字
4、对象==布尔 都转为数字
5、字符串==数字 字符串转为数字
6、字符串==布尔 都转为数字
7、数字==布尔 布尔转为数字
8、null==undefined true
9、null===undefined false
10、NaN==NaN rd false
