##DOM节点
Node 在页面中出现的所有东西都是节点，元素，注释，文本都是节点
应用场景：通过节点可以进行DOM对象的增 删 改 查

节点类型 node Type nodeName nodeValue
元素节点(元素标签) 1 大写的标签 null
文本节点(文字) 3 #text 文字内容
注释节点(注释) 8 #comment 注释内容
document(整个文档) 9 #document null

##DOM节点的增删改查

###创建元素 节点
createElement() 创建 创造

####将新创建的元素节点添加到指定元素所有内容之后(末尾)
oDiv.parentNode.appendChild( oDiv);

appendChild前是oDiv的父级节点

####将创建的元素添加到已有元素之前
父级元素.insertBefore（要放置的元素，放在谁之前）
oldTag.parentNode.insertBefore(newTag , oldTag)

###创建属性节点
setAttribute(属性名，属性值)—–直接向对象添加一对键值对
这个属性会在html标签中出现

oDiv.setAttribute(‘zhufeng’,’peixun’);
oDiv.getAttribute(‘zhufeng’,’peixun’);
oDiv.removeAttribute(‘zhufeng’,’peixun’);

###删除创建的元素节点
父级元素.removeChild(删除的对象)
op.parentNode.removeChild(op)
已有的和新创建的都可以删除

###克隆元素节点
op.cloneNode(true);
默认参数为false(不填参数)：只克隆当前元素
true 克隆当前元素与元素的所有子节点

###替换元素节点
父级元素.replaceChild(替换的对象，被替换的对象)
op.parentNode.replaceChild(替换的对象，被替换的对象)
任何元素都可以被替换

###math方法及应用

1、Math.abs() 取绝对值
2、Math.ceil() 向上取整（出现小数的就向上+1）
3、Math.floor() 向下取整
4、Math.round() 四舍五入
5、Math.max(val1,val2, val3….) 取最大值
6、Math.min(val1,val2,val3,….) 取最小值
7、Math.random()获取[0-1]之间的随机小数（不包含1）;
8、Math.round(Math.random()*(m-n)+n) 获取任意两个数之间的随机数
9、Math.PI 3.14159265..
10、Math.pow n的m次幂

parentNode 获取父级元素节点

previousElementSibling 获取上一个元素节点
nextSibling 获取下一个节点
nextElementSibling 获取下一个元素节点
firstChild 获取第一个节点
lastChild 获取最后一个节点
children 获取所有子元素节点
childNodes 获取所有子节点
