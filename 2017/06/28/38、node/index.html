##什么是node？
node就是一个运行环境，可以使js运行在服务端 浏览器不能操作本地文件

##node

  • 非阻塞异步I/O
  • commonjs 规范 通过require引入module.exports 导出
  • 基于 v8 引擎
  • npm 包管理器（社区有后台的包还有前端的一些包） bower也是包管理器
    ##node的安装

Mac 安装 Homebrew

##配置系统环境变量
右键计算机属性—>高级系统设置—>环境变量—>系统变量—>配置path—>node安装路径

##进入交互模式解析器

  • 命令行中输入 node
  • Ctrl+c 点击两次退出
