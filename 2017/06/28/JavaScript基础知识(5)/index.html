##Array数组

什么方法改变原数组，什么方法原数组不变
什么方法返回查找项或者删除项，什么方法返回长度

###删除数组末尾项的方法

  • pop()
  • ary.length–;
  • ary.splice(ary.length-1,1)
  • ary.splice(ary.length-1)

###向数组末尾添加

  • push
  • ary.splice(ary.length-1,0,x)

###向数组开始增加

  • unShift()
  • ary.splice(0,0,x);

##字符串 string

字符串是JS中代表字符串的变量，字符串用引号包裹，单引号双引号都可以

字符串是有长度有索引的

  • 获取指定索引位置的字符 charAt(索引)
  • charCodeAt(索引)–>获取指定索引位置字符对应的ASCII值（Unicode编码）请百度

###字符串的截取

  • substr(n,m) 从索引n开始，截取m个字符
    • 原字符串不改变，返回查找的值
  • substring(n,m) 从索引n开始找到索引m处，不包含m
  • slice（n,m) 从索引n开始找到索引m处，不包含m—>支持负数为索引

###查找字符所在字符串的索引
兼容所有浏览器

  • indexOf:返回该值第一次出现所在位置的索引 返回值为-1 则说明没有这 个字符
  • LastIndexOf:返回该值最后一次出现所在位置的索引 返回值为-1 则说明没有这个字符
  • search : 用来查找字符的，找到返回对应字符串的索引 找不到返回-1；
  • match（‘value’） 匹配字符的，返回的是一个数组，数组中是匹配到的字符，找不到-1

返回值为-1 则说明没有这个字符

字符串中字母的大小写转换

toLowerCase() 转化为小写
toUpperCase() 转化为大写

替换与正则表达式匹配的子串

replace（oldWord，newWord）
在不使用正则的情况下，每一次调用方法只能替换一次 一般

###将字符串按照指定分隔符产生成数组
split（分隔符）
split() 将字符串当做一项转为数组
split(‘’)将字符串的每一项都当做数组中的每一项分割符要在字符串中存在 比较分割符分割后不会出现

##定时器
传递的参数，第一个参数为方法（也就是一个函数） 第二个为参数为时间

###指定一个时间，重复一个方法
Windows.setInterval([function],[interval])

###指定一个时间，达到时间后执行一个方法，只执行一次
windows.settimeout([function],[interval])

定时器的返回值是一个数字，代表我市这个文档下第几个定时器

####清除定时器

  • clearTimeout()
  • clearinterval()
