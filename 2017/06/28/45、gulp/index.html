##安装gulp

  • npm install -g gulp
  • npm install gulp -S
  • npm install gulp-less gulp-concat gulp-uglify gulp-rename gulp-clean-css gulp-minify-html gulp-inject gulp-babel gulp-plumber gulp-connect –save-dev
  • npm install babel-preset-es2015 babel可以把es6转es5
  • npm install gulp-load-plugins 这个插件可以帮你加载所有的gulp插件
